Getting started

YAKINDU Statechart Tools is an integrated modeling environment for the specification and development of reactive, event-driven systems based on the concept of statecharts. It comes with a graphical editor and powerful simulation and code generation features. Statecharts can be defined independently of any concrete target language. Alternatively, you can integrate them with your C / C++ code.
